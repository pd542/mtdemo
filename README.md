# FuturesSimTrader 期货模拟交易安卓 App

一个按用户截图风格制作的 Android 原生 Kotlin + Jetpack Compose 工程。

## 功能

- 五个底部页面：行情 / 图表 / 交易 / 历史 / 信息
- 国内期货品种示例：螺纹钢、沪金、豆粕、沪铜、沪深300股指
- 行情适配：优先请求新浪财经免费期货行情 `https://hq.sinajs.cn/list=nf_...`，失败自动降级为本地模拟行情
- M5 K线图：蜡烛图、网格线、均线、当前价线
- 模拟交易：市价、限价、止损委托；买开/卖开/买平/卖平；持仓、委托、成交、拒单、撤单；保证金、浮盈、权益、风险度

## 构建

需要本地安装 Android Studio 或 Android SDK + JDK 17。

```bash
cd FuturesSimTrader
./gradlew assembleDebug
```

如果没有 Gradle Wrapper，可在 Android Studio 中直接打开本目录，或安装 Gradle 后执行：

```bash
gradle assembleDebug
```

## 注意

本 App 仅用于模拟交易，不连接真实期货柜台，不会发送真实订单。免费行情接口可能不稳定，代码已经设计为失败后自动使用本地模拟行情，保证界面可运行。

## 本次图表页订单工具二次更新

- SL/TP 不再默认显示：点击底部 `SL` 或 `TP` 按钮才显示对应线；再次点击会隐藏并取消该设置。
- 挂单线根据当前市场价自动切换买卖方向：
  - 限价模式：挂单价高于市场价为卖出限价，低于市场价为买入限价。
  - 止损模式：挂单价高于市场价为买入止损，低于市场价为卖出止损。
  - 止损限价模式同理按买入/卖出方向切换。
- 拖动挂单线越过市场价时，TP/SL 会自动保持在合理一侧。
- 底部订单框支持上拉展开成大半屏下单票据，不跳转页面。
- 展开票据包含交易量、滑杆、预付款/可用、价格输入、价格/点数/USD切换、SL/TP输入、超期、评论、下单按钮。
- 价格/SL/TP 输入与图表线双向同步。
